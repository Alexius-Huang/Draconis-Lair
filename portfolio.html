Slippy Snake

CodePen Project HTML5 CSS3 Bootstrap JavaScript HammerJS MousetrapJS jQuery

Snake Game is one of the simpliest to make. The game can also play on mobile and it automatically adjust the grid system to appropriate size. You can use the arrow keys to control the snake or swipe the mobile to control.

In order to render out the whole snake body parts, I store the head position respect to the grid system and another array for storing the body parts of the snake.

It is easy to check if the snake collide the border by using the current direction of the snake and check the next position whether collide the border or not. However, in order to check if it collide the snake itself, I also used another two-dimensional array to record the distribution of the snake part and compare to the next position of the snake.

Lastly, apple (which is represented by red square) is generated randomly and when the snake eat it, the snake will become longer.

I put the project and the code on my site, CodePen and my GitHub page.

Brick Breaker

CodePen Project HTML5 CSS3 Bootstrap JavaScript jQuery

Brick Breaker is difficult to make mainly because the collision detection is a little bit tricky. I construct the game with the grid or coordinate system and defined unit width and height of the brick. Bricks can be longer or thicker and it can have more than single lives.

By using the position and the direction of the ball with the corresponding grid system, it can calculate or verify the occurence of the collision. When the ball collide with the brick, the brick will decrease life or vanished and the ball will be reflected to another direction.

You can use the mouse to control the horizontal position of the paddle and the ball collide position on the paddle will also affect the angle of refection of the ball.

Tetris

CodePen Project HTML5 CSS3 Bootstrap JavaScript jQuery HammerJS MousetrapJS

Tetris is a popular game among several decades. It's a kind of tile-matching game which you can control the block, rotate and landing the block. When there's at least a row is filled with block, the row will be cleared out and being filled by the upper rows.

In this CodePen project, you can play with arrow keys and space.

- Up arrow can rotate the block clockwisely
- Down / Left / Right arrow moves the Tetris block down / left / right per block
- Space key lands down the Tetris block immediately

You can also use mobile to play the game (but it would be a little tough)

- Swipe upward will rotate the Tetris block clockwise - Swipe Down / Left / Right will move the Tetris block down / left / right per block - Double taps lands the Tetris block immediately

There are more details to tell. The algorithm in rotating the Tetris block use a technique called Linear Transformation. Simply, it can convert a 2-D (even 3-D) plane coordination and transform into another 2-D plane with matrix arithmatics. In other words, the rotation can be achieved by transforming into another plane coordination.

I put the project and the code on my site, CodePen and my GitHub page.

Simple Harmonic Motion : Box-String System

CodePen Project Physics HTML5 CSS3 Bootstrap JavaScript jQuery HammerJS

It is widely used in mechanical engineering for describing the motion of a pendulum or box-spring system and etc. In electrical engineering, it can also illustrate how capacitors or inductors behavior in resonance or oscillation circuits.

Game of Life

CodePen Project Conway's Game of Life Mathematic HTML5 CSS3 JavaScript jQuery Bootstrap

Game of Life is a kind of cellular automation that simulates the evolution of the cell patterns and results in many types of interesting patterns. For more information, you can look at Conway's Game of Life Wiki.

In Game of Life, we can arrange the pattern of living cells in a two dimensional plane. There are basic rules in Game of Life that measures the relationship between the main cells and eight neighbouring cells:

- Too much neighbour cells (bigger than or equal to 4) that cause overpopulation which makes the main cell die.
- Too less neighbor cells (smaller than or equal to 1) that which cause the death of the main cell due to isolation and lacking nutrition
- Only moderate condition, which there are only 2 or 3 neighbour cells, that the main cell live on to next generation
- The dead cell will revive when there are 3 neighbour cells surrounded

The algorithm is quite simple and I put it on my site, CodePen and my GitHub page. I also wrote articles about Game of Life in Medium.
